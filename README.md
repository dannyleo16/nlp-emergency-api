# NLP Emergency API

API for Named Entity Recognition (NER) on Spanish emergency reports using a fine-tuned BERT model.

---

## Model

Hugging Face model:

https://huggingface.co/dannyLeo16/ner_model_bert_base

---

## Example

Input text:

Hay una persona herida en la avenida Loja frente al hospital.

Output:

PER → persona  
LOC → avenida Loja
